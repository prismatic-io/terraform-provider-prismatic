# prismatic_organization_roles (Data Source)

Data source to list Prismatic Organization Roles.

Use this data source to retrieve the list of available organization roles in Prismatic. This is useful for assigning roles to organization users.

## Example Usage

```terraform
# Get all organization roles
data "prismatic_organization_roles" "roles" {}

# Find the Admin role
locals {
  admin_role = [for r in data.prismatic_organization_roles.roles.roles : r if r.name == "Admin"][0]
}

# Output role information
output "admin_role_id" {
  value = local.admin_role.id
}

output "all_roles" {
  value = data.prismatic_organization_roles.roles.roles
}
```
