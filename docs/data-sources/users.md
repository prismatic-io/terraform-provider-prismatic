# prismatic_users (Data Source)

Data source to list Prismatic Users.

Use this data source to retrieve the list of users in Prismatic. By default, it returns only organization users (customer_is_null = true).

## Example Usage

```terraform
# Get all organization users
data "prismatic_users" "org_users" {
  customer_is_null = true
}

# Get all customer users
data "prismatic_users" "customer_users" {
  customer_is_null = false
}

# Output user information
output "org_user_count" {
  value = length(data.prismatic_users.org_users.users)
}

output "org_users" {
  value = data.prismatic_users.org_users.users
}
```
